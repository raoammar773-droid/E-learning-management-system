# E-Learning Management System (Virtual Classroom)

A comprehensive, console-based Virtual Classroom Management System developed in C++ using Object-Oriented Programming (OOP) principles. This system bridges the gap between students and teachers, simulating a real-world academic portal.

## 🚀 Features

### For Teachers:
* **Class Management:** Ability to schedule and host online sessions.
* **Assessments:** Create and assign quizzes and homework/assignments to the class.
* **Logs & Tracking:** Automated tracking of teacher logs for course completion.

### For Students:
* **Academic Hub:** Access and attend scheduled virtual classes.
* **Submissions:** Attempt assigned quizzes and digitally submit assignments.

## 🛠️ Concepts Demonstrated
* **Object-Oriented Programming (OOP):** Robust use of Classes, Inheritance, and Encapsulation to manage `User`, `Teacher`, and `Student` modules.
* **File Handling in C++:** Implementation of persistent data storage using file streams (`ofstream`) to securely log user actions and preserve application state across restarts.
